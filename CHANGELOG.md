# Changelog

This changelog tracks all significant changes to DevFlow.

The format of this changelog is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-27

### Added

- Root agent implementation using Google ADK
- Read-only GitHub integration via MCP
- Support for Gemini 2.5 Flash
- Interactive command-line interface using natural language
- In-memory session service
- Configuration via `.env` file
- Automated evaluation using pytest
- Problem statement and solution overview
